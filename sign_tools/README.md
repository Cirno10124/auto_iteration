
  # Web Annotation Tool Design

  This is a code bundle for Web Annotation Tool Design. The original project is available at https://www.figma.com/design/ZZthuEr0mTFtxo5KKNDj4o/Web-Annotation-Tool-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## 音频标注工具后端（annotation_server）

  本仓库内另有一个 Node 后端 + 静态前端的标注服务，位于子目录 `annotation_server/`，便于单独打包与部署。进入该目录后执行 `npm install` 与 `npm run dev` 即可启动标注 API 与页面，详见 `annotation_server/README.md`。

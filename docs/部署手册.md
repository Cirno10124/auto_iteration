# 部署手册（auto_iteration）

## 1. 本地 Python 部署

### 1.1 环境准备

- Python 3.11
- 可选 GPU + CUDA（训练建议）

### 1.2 安装依赖

```bash
pip install -r requirements.txt
```

### 1.3 配置与启动

```bash
python orchestrator.py --env dev
```

或指定配置文件：

```bash
python orchestrator.py --config orchestrator_config.prod.json
```

### 1.4 敏感信息注入

```bash
set HF_HUB_TOKEN=your_token
```

## 2. GPU 容器部署（推荐）

### 2.1 前提

- Docker
- NVIDIA Container Toolkit
- 主机可执行 `nvidia-smi`

### 2.2 构建镜像

```bash
docker compose -f docker-compose.gpu.yml build
```

### 2.3 GPU 健康检查

```bash
docker compose -f docker-compose.gpu.yml run --rm orchestrator-gpu python3 scripts/gpu_health_check.py
```

### 2.4 启动服务

```bash
docker compose -f docker-compose.gpu.yml up
```

## 3. 部署后检查项

- 日志文件是否生成（`logging.log_dir`）
- 是否有 `speaker/iter/step/model` 上下文字段
- 训练输出目录是否正常写入（`model_dir`）
- 评估与转换产物是否生成（`manifests` / `ggml_dir`）
